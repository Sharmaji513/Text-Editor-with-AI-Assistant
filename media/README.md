# Media Folder

This folder contains demo videos and other media assets for the project.

## Files

- `demo-video.mp4` - Demo video showcasing the collaborative editor features

## Usage

The video is referenced in the main README.md file. To view it, click the link in the README or download the file directly.

## Note

If the video file is large (>50MB), consider using Git LFS:
```bash
git lfs install
git lfs track "*.mp4"
git add .gitattributes
```
